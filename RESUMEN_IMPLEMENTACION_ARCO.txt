================================================================================
FLUJO DE EJECUCIÓN ACTUALIZADO
================================================================================

Cada iteración k del control:

1. Estado actual: X[:, k] + s_current
2. Predicción del arco (horizonte N):
   Para j = 0 a N:
     s_pred_j = s_current + j * dt * s_nominal_rate
     pos_ref_j = position_by_arc_length(s_pred_j)  ← Mapeo dinámico
     quat_ref_j = quaternion_by_arc_length(s_pred_j)
     vel_ref_j = velocity_by_arc_length(s_pred_j)
     X_d_j = [quat_ref_j, pos_ref_j, velocidades]

3. MPC resuelve OCP:
   min_u J = ||error|| + costo_control + costo_arco
   Salidas: u_opt = [F, τx, τy, τz, u_s_opt]

4. Integración:
   X[:, k+1] = RK4(X[:, k], u_opt[0:4])  ← Solo dinámicas de pose
   s_current = s_current + u_s_opt * dt  ← Integración de arco

5. Siguiente iteración usa nuevo s_current
